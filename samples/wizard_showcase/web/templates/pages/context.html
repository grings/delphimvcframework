Request context

A single TContextInfo record carries [MVCFromHeader] and [MVCFromCookie] attributes on its fields. Each request resolves both sources in a single argument.

User-Agent (header)
{{:user_agent}}
Accept-Language (header, default "en")
{{:language}}
Theme (cookie, default "light")
{{:theme}}

The handler also writes the theme cookie on every response. Open DevTools → Application → Cookies, change the value (e.g. to dark), reload — the new value flows back through [MVCFromCookie('theme', 'light')] into the record. Clear the cookie to see the default kick in.
